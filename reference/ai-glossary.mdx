## AI

Artificial Intelligence. Computer system က လူလိုနားလည်/ရေး/ခွဲခြမ်း/ဆုံးဖြတ်သလိုလုပ်နိုင်တဲ့နည်းပညာအုပ်စုပါ။

## Agent

Goal တစ်ခုပေးလိုက်ရင် tool တွေသုံးပြီး step-by-step လုပ်ဆောင်နိုင်တဲ့ AI workflow ပုံစံပါ။

## Context window

AI model က တစ်ခါတည်းမှတ်ထားနိုင်တဲ့စာပမာဏပါ။

## Embedding

Text/image စတဲ့ data ကို model နားလည်နိုင်တဲ့ number representation အဖြစ်ပြောင်းထားတာပါ။

## Hallucination

AI က မမှန်တာကို မှန်သလို confidence နဲ့ပြောတာပါ။

## LLM

Large Language Model. စာသားနားလည်ပြီး response generate လုပ်နိုင်တဲ့ AI model အမျိုးအစားပါ။

## Prompt

AI ကိုပေးတဲ့ instruction/context ပါ။

## RAG

Retrieval-Augmented Generation. External documents/database ကနေ relevant info ရှာပြီး AI response ထဲထည့်သုံးတဲ့နည်းပါ။

## Token

AI model က text ကိုဖတ်တဲ့ unit ပါ။ Word သို့မဟုတ် word အပိုင်းအစလိုယူဆနိုင်ပါတယ်။
