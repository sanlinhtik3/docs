## Address

Crypto လက်ခံ/ပို့ဖို့သုံးတဲ့ wallet account number လိုမျိုး string ပါ။

## Blockchain

Transaction history တွေကို public ledger အဖြစ်မှတ်ထားတဲ့ network system ပါ။

## CEX

Centralized Exchange. Binance, Coinbase လို company ကစီမံတဲ့ exchange ပါ။

## DEX

Decentralized Exchange. Wallet နဲ့ connect လုပ်ပြီး smart contract မှတစ်ဆင့် swap လုပ်တဲ့ exchange ပါ။

## DCA

Dollar-Cost Averaging. ဈေးတက်တက်ကျကျ သတ်မှတ်ထားတဲ့ပမာဏကို ပုံမှန်ဝယ်စုတဲ့နည်းပါ။

## DeFi

Decentralized Finance. Blockchain ပေါ်က lending, borrowing, swapping စတဲ့ financial apps တွေပါ။

## Gas fee

Blockchain transaction လုပ်တဲ့အခါ network ကိုပေးရတဲ့ fee ပါ။

## Seed phrase

Wallet ကိုပြန်ဖွင့်နိုင်တဲ့ recovery words ပါ။ ဘယ်သူ့ကိုမှမပေးရပါ။

## Stablecoin

USD လို fiat currency တန်ဖိုးနဲ့ကပ်ထားဖို့ရည်ရွယ်တဲ့ crypto asset ပါ။

## Tokenomics

Token supply, allocation, unlock, incentive system တွေကိုခေါ်ပါတယ်။

## Wallet

Crypto address/private key တွေကိုထိန်းတဲ့ app/device/system ပါ။
