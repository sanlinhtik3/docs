---
title: "Prompt Library"
description: "Prompt bank မဟုတ်ဘဲ daily work အတွက် reusable operating prompts စုစည်းမှု။"
---

ဒီ page က copy-paste prompt တွေစုထားတာပါ။ ဒါပေမယ့် prompt bank သက်သက်မဟုတ်ပါ။ Prompt တိုင်းမှာ role, goal, context, output, quality gate ပါတဲ့ operating prompt ဖြစ်အောင်ရေးထားပါတယ်။

## 1. Universal Task Operator

```text
Act as my task operator.

Goal:
[desired outcome]

Context:
[background, audience, constraints]

Inputs:
[notes, draft, links, examples]

Output format:
[checklist/table/markdown/script/code]

Process:
1. Restate the task.
2. Identify missing info.
3. Produce the best first version.
4. Run the quality checks.
5. Give next actions.

Quality checks:
- Accurate
- Practical
- Concise
- Safe
- Ready to use
```

## 2. Rewrite Without Losing Voice

```text
Rewrite this draft without losing my voice.

Voice:
Friendly Burmese + English tech terms mix.

Improve:
- Clarity
- Structure
- Flow
- Practical value
- Conciseness

Do not:
- Add fake facts
- Make it sound corporate
- Remove useful local examples

Draft:
[paste draft]
```

## 3. Make It Beginner-Friendly

```text
Explain this for a Myanmar beginner.

Topic:
[topic]

Requirements:
- Burmese explanation
- English tech terms in parentheses
- One simple analogy
- One practical example
- Mistakes to avoid
- Short checklist
```

## 4. Turn Notes Into Operating Manual

```text
Turn these rough notes into an operating manual page.

Audience:
[target user]

Structure:
- TL;DR
- When to use
- Core idea
- Workflow
- Copy-paste prompt/template
- Example
- Quality check
- Safety notes
- Takeaway

Notes:
[paste notes]
```

## 5. Review My Output

```text
Review this output before I publish it.

Check:
1. Accuracy
2. Clarity
3. Practical usefulness
4. Hype/overclaiming
5. Safety/privacy risks
6. Missing context
7. Stronger title/hook options

Return:
- Pass/fail
- Top 5 fixes
- Improved version if needed
```

## 6. Build My Content System

```text
Act as my content system operator.

Niche:
[niche]

Audience:
[audience]

Platforms:
[platforms]

Create:
1. 5 content pillars
2. 20 content ideas
3. Weekly posting plan
4. Repurpose workflow
5. Quality checklist
6. One sample post
```

<Tip>
Prompt ကို copy-paste လုပ်ပြီးတာနဲ့မပြီးသေးပါ။ Bracket ထဲက `[ ]` တွေကို ကိုယ့် context နဲ့ပြည့်အောင်ဖြည့်မှ output ကတန်ဖိုးရှိပါတယ်။
</Tip>
