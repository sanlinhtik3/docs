Content Creator Operator ဆိုတာ idea တစ်ခုကို publish-ready content system အဖြစ်ပြောင်းပေးတဲ့ workflow ပါ။ AI ကို “post ရေးပေး” လို့မခိုင်းဘဲ **research → angle → hook → draft → repurpose → review** ခိုင်းပါ။

## TL;DR

- Content က prompt တစ်ခုနဲ့မပြီးပါ။ Angle, audience, format, platform, review loop လိုပါတယ်။
- Hook က attention ရယူတယ်၊ body က value ပေးတယ်၊ CTA က action ပေးတယ်။
- AI ကို viral hype မဟုတ်ဘဲ clarity + usefulness + trust ထုတ်ခိုင်းပါ။
- Creator voice ကို prompt ထဲမှာသတ်မှတ်ပါ။

## Creator OS Loop

<Steps>
  <Step title="Capture">
    Idea, story, lesson, question, audience pain point ကိုစုပါ။
  </Step>
  <Step title="Research">
    Fact, source, examples, objections ကိုစစ်ပါ။
  </Step>
  <Step title="Angle">
    Topic ကို unique viewpoint တစ်ခုအဖြစ်ချုံ့ပါ။
  </Step>
  <Step title="Draft">
    Platform-specific format နဲ့ရေးပါ။
  </Step>
  <Step title="Repurpose">
    Long video/script ကို short post, Telegram update, carousel, newsletter အဖြစ်ပြောင်းပါ။
  </Step>
  <Step title="Review">
    Accuracy, tone, hook, usefulness, CTA, risk ကိုစစ်ပါ။
  </Step>
</Steps>

## Master Content Prompt

```text
Act as my senior content operator.

Topic:
[topic]

Audience:
[ဘယ်သူအတွက်လဲ]

Platform:
[YouTube / Facebook / Telegram / TikTok / Newsletter]

Voice:
Friendly Burmese + English tech terms mix. Practical, clear, no hype.

Goal:
[educate / persuade / explain / announce / build trust]

Inputs:
[rough notes, links, story, examples]

Create:
1. One sharp content angle
2. 5 hooks
3. Main draft
4. CTA
5. Risk/fact-check notes
6. 3 repurpose ideas

Quality gate:
- Useful in real life
- Clear for beginner
- No fake claim
- No overpromise
- Sounds like a human creator
```

## YouTube Script Operator

```text
Turn this topic into a YouTube script.

Structure:
1. Hook: first 10 seconds
2. Problem: why viewers care
3. Core lesson: 3-5 sections
4. Example: Myanmar/local context
5. Mistakes to avoid
6. Summary
7. CTA

Style:
Conversational Burmese, with English tech terms where useful.
Keep paragraphs short for speaking.
```

## Repurpose Operator

```text
Repurpose this long content into:
1. Facebook post
2. Telegram short update
3. 5 short-video hooks
4. 3 carousel slide outlines
5. Newsletter summary

Keep the same core idea.
Do not invent new facts.
Make each platform feel native.
```

## Creator Quality Check

- Hook က clear လား
- Reader/viewer ကို ၁ မိနစ်အတွင်း value ပေးလား
- Claim တွေကို fact-check လုပ်နိုင်လား
- CTA က natural ဖြစ်လား
- Hype များနေသလား
- Creator voice မပျောက်ဘူးလား

<Tip>
Top creator prompt ဆိုတာ magic word မဟုတ်ပါ။ Audience, angle, voice, platform, quality gate ကိုတိတိကျကျပေးတဲ့ operating prompt ပါ။
</Tip>
