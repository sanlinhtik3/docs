Blockchain ဆိုတာ transaction တွေကို အတန်းလိုက်မှတ်ထားတဲ့ public ledger တစ်မျိုးပါ။ Ledger ဆိုတာ စာရင်းစာအုပ်လို့မှတ်လို့ရပါတယ်။

Bank မှာ transaction history ကို bank server ကထိန်းထားတတ်ပါတယ်။ Blockchain မှာတော့ network ထဲက computer တွေက transaction history ကို စစ်ပြီးသဘောတူထားပါတယ်။

## Transaction ဆိုတာဘာလဲ

Transaction ဆိုတာ crypto asset တစ်ခုကို address တစ်ခုကနေ address တစ်ခုကိုရွှေ့တာပါ။

ဥပမာ:

- ကိုယ့် Binance account ကနေ wallet ထဲ USDT ပို့တာ
- Wallet ကနေ တခြားလူရဲ့ wallet ကို USDC ပို့တာ
- DEX မှာ token swap လုပ်တာ

## Block ဆိုတာဘာလဲ

Block ဆိုတာ transaction တွေကို စုပြီးမှတ်ထားတဲ့ package လို့မြင်လို့ရပါတယ်။ Block တစ်ခုနဲ့တစ်ခု ဆက်နေတဲ့အတွက် blockchain လို့ခေါ်တာပါ။

## Network fee / gas fee ဆိုတာဘာလဲ

Crypto transaction လုပ်တဲ့အခါ network ကိုအသုံးပြုတဲ့ fee ပေးရတတ်ပါတယ်။

- Ethereum မှာ gas fee လို့ခေါ်တာများပါတယ်။
- Tron, BNB Chain, Solana စတာတွေမှာလည်း fee ရှိပါတယ်။
- Fee ပမာဏက network traffic ပေါ်မူတည်ပြီးပြောင်းတတ်ပါတယ်။

<Warning>
Fee နည်းတယ်ဆိုတာနဲ့ network တစ်ခုကို blind မရွေးပါနဲ့။ Recipient က အဲဒီ network ကို support လုပ်လားဆိုတာအရင်စစ်ပါ။
</Warning>

## Blockchain transaction က ပြန်ဖျက်လို့ရလား

များသောအားဖြင့် blockchain transaction က confirmed ဖြစ်သွားရင် reverse လုပ်လို့မရပါ။ ဒါကြောင့် crypto မှာ “မပို့ခင်စစ်” ဆိုတာအရေးကြီးပါတယ်။

## မပို့ခင်စစ်ရမယ့် ၅ ချက်

- Address မှန်လား
- Network မှန်လား
- Amount မှန်လား
- Fee လက်ခံနိုင်လား
- Test transaction လုပ်ပြီးပြီလား
