Crypto မှာ beginner တွေ အများဆုံးမှားတာက exchange, wallet, network သုံးခုကို တစ်ခုတည်းလိုထင်တာပါ။ တကယ်တော့ မတူပါ။

## Exchange ဆိုတာဘာလဲ

Exchange ဆိုတာ coin ဝယ်ရောင်းလုပ်တဲ့ platform ပါ။ ဥပမာ Binance, Coinbase, OKX စတာတွေ။

Exchange ထဲမှာ asset ထားရင် သုံးရလွယ်ပါတယ်။ ဒါပေမယ့် account security, KYC, withdrawal rules, freeze risk တွေရှိပါတယ်။

## Wallet ဆိုတာဘာလဲ

Wallet ဆိုတာ blockchain address တွေကိုထိန်းတဲ့ tool ပါ။ Self-custody wallet ဆိုရင် seed phrase/private key ကို ကိုယ်တိုင်ထိန်းရပါတယ်။

ဥပမာ:

- MetaMask
- Rabby
- Trust Wallet
- Phantom
- Hardware wallet

<Warning>
Seed phrase ကို သိတဲ့သူက wallet ထဲက asset ကိုထိန်းနိုင်ပါတယ်။ Seed phrase ကို ဘယ်သူ့ကိုမှမပေးပါနဲ့။ AI ထဲလည်းမထည့်ပါနဲ့။
</Warning>

## Network ဆိုတာဘာလဲ

Network ဆိုတာ blockchain rail ပါ။ USDT တစ်ခုတည်းကို Ethereum, Tron, BNB Chain စတဲ့ network မျိုးစုံပေါ်မှာရှိနိုင်ပါတယ်။

ပို့တဲ့သူနဲ့လက်ခံမယ့်သူ network တူရပါမယ်။

## Beginner example

ကိုယ်က Binance ကနေ USDT withdraw လုပ်မယ်ဆိုရင်:

1. Recipient address ကိုယူပါ။
2. Recipient က ဘယ် network လက်ခံလဲစစ်ပါ။
3. Binance withdraw page မှာ အဲဒီ network ကိုတိတိကျကျရွေးပါ။
4. Small test amount အရင်ပို့ပါ။
5. ရောက်ပြီးမှ amount ကြီးပို့ပါ။

## ဘယ်အချိန်မှာ exchange ထဲထားမလဲ

- Trading လုပ်နေတဲ့ amount
- Short-term buy/sell လုပ်မယ့် amount
- Beginner အနေနဲ့ self-custody မကျွမ်းသေးတဲ့ small amount

## ဘယ်အချိန်မှာ wallet ထဲထားမလဲ

- Long-term hold
- DeFi app သုံးမယ့် asset
- Exchange risk ကိုလျှော့ချင်တဲ့ amount
