Crypto ဆိုတာ internet ပေါ်မှာ သုံးလို့ရတဲ့ digital asset အမျိုးအစားပါ။ Bank account ထဲက number လိုမျိုးပဲ မြင်ရပေမယ့် crypto ရဲ့အဓိကကွာခြားချက်က ownership ကို blockchain network ပေါ်မှာ စစ်လို့ရတာပါ။

ရိုးရိုးပြောရရင် Crypto က “internet-native money and assets” လို့မှတ်လို့ရပါတယ်။

## Crypto ကို ဘာအတွက်သုံးလဲ

Crypto ကို လူတွေသုံးတဲ့ use case တွေက အများကြီးရှိပါတယ်။

- Value ကို သိမ်းထားဖို့
- Stablecoin နဲ့ dollar-like balance ထားဖို့
- နိုင်ငံတကာ transfer လုပ်ဖို့
- DeFi app တွေသုံးဖို့
- Game/NFT/community asset တွေကိုင်ဖို့
- Investment/trading လုပ်ဖို့

ဒီ Docs ရဲ့ crypto direction က speculation တစ်ခုတည်းမဟုတ်ပါ။ **Crypto ကို bank လိုမျိုး လုံခြုံစွာသုံးတတ်ဖို့** ကို အရင်ဦးစားပေးပါတယ်။

## Coin, token, stablecoin က ဘာကွာလဲ

- **Coin**: Network ရဲ့ native asset ပါ။ ဥပမာ Bitcoin network မှာ BTC, Ethereum network မှာ ETH။
- **Token**: Existing blockchain ပေါ်မှာထုတ်ထားတဲ့ asset ပါ။ ဥပမာ Ethereum ပေါ်က USDT token။
- **Stablecoin**: USD လို fiat currency တန်ဖိုးနဲ့ကပ်ထားတဲ့ crypto asset ပါ။ ဥပမာ USDT, USDC။

<Tip>
Beginner အတွက် “coin/token” ခွဲခြားတာထက် ပိုအရေးကြီးတာက ကိုယ်ပို့မယ့် network မှန်လား၊ address မှန်လား၊ recipient မှန်လားဆိုတာပါ။
</Tip>

## Crypto က bank နဲ့ ဘာကွာလဲ

Bank မှာ account access ကို bank ကထိန်းပေးပါတယ်။ Password မေ့ရင် reset လုပ်လို့ရတတ်ပါတယ်။ Crypto wallet မှာတော့ self-custody ဆိုရင် seed phrase ကို ကိုယ်တိုင်ထိန်းရပါတယ်။

အဲဒါကြောင့် crypto မှာ freedom ပိုရှိသလို responsibility လည်းပိုကြီးပါတယ်။

## Beginner checklist

Crypto စမသုံးခင် ဒီအချက်တွေကိုနားလည်အောင်လုပ်ပါ။

- Exchange နဲ့ wallet က မတူဘူး။
- Address မှားရင် transaction ပြန်ဖျက်လို့မရတတ်ဘူး။
- Network မှားရင် asset ပျောက်နိုင်တယ်။
- Seed phrase ကို screenshot မရိုက်ရ။
- Scam link ကို wallet connect မလုပ်ရ။
- Test transaction အရင်လုပ်ရ။
