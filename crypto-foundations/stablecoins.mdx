Stablecoin ဆိုတာ USD လို fiat currency တစ်ခုနဲ့ တန်ဖိုးကပ်ထားဖို့ရည်ရွယ်တဲ့ crypto asset ပါ။ Beginner တွေ Myanmar context မှာ အများဆုံးတွေ့မယ့် stablecoin က USDT ဖြစ်တတ်ပါတယ်။

## Stablecoin ကို ဘာကြောင့်သုံးကြလဲ

Crypto market ထဲမှာ stablecoin က dollar balance လိုမျိုးအသုံးဝင်ပါတယ်။

- Coin ဝယ်မယ့်အချိန်စောင့်ဖို့
- Exchange တွေကြား value ရွှေ့ဖို့
- P2P မှာ buy/sell လုပ်ဖို့
- Volatile coin ထဲမထားချင်တဲ့အချိန် cash-like position ထားဖို့

## USDT, USDC က ဘာကွာလဲ

USDT နဲ့ USDC နှစ်ခုလုံးက USD တန်ဖိုးနဲ့ကပ်ထားဖို့ရည်ရွယ်ထားတဲ့ stablecoin တွေပါ။ Issuer, reserve, supported networks, exchange support တွေကွာပါတယ်။

<Info>
Stablecoin တစ်ခုက USD နဲ့ကပ်ထားတယ်လို့ဆိုပေမယ့် risk လုံးဝမရှိဘူးလို့မဆိုလိုပါ။ Issuer risk, regulation risk, depeg risk, network risk တွေရှိနိုင်ပါတယ်။
</Info>

## Stablecoin risk

သတိထားရမယ့်အချက်တွေ:

- Stablecoin တန်ဖိုးက $1 ကနေယာယီကွာနိုင်တယ်။
- တချို့ network တွေမှာ fake token တွေရှိနိုင်တယ်။
- Wrong network နဲ့ပို့ရင် asset ပျောက်နိုင်တယ်။
- Exchange account freeze, P2P dispute စတာတွေဖြစ်နိုင်တယ်။

## Network အမျိုးမျိုးပေါ်က stablecoin

USDT တစ်ခုတည်းပဲဖြစ်ပေမယ့် network မတူရင် မတူတဲ့ asset flow ဖြစ်နိုင်ပါတယ်။

ဥပမာ:

- USDT on Ethereum
- USDT on Tron
- USDT on BNB Chain
- USDT on Solana

<Warning>
USDT address တူသလိုထင်ရပေမယ့် network မတူရင် transaction ပျောက်နိုင်ပါတယ်။ Recipient က ဘယ် network support လုပ်လဲ မေးပြီးမှပို့ပါ။
</Warning>

## Beginner rule

Stablecoin ပို့တဲ့အခါ amount ကြီးမပို့ခင် $1 or small amount test transaction အရင်လုပ်ပါ။ Test transaction ရောက်ပြီးမှ amount ကြီးကိုပို့ပါ။
