Exchange account က crypto bank account လိုဖြစ်နေနိုင်ပါတယ်။ ဒါကြောင့် password တစ်ခုတည်းနဲ့မထားသင့်ပါ။

## မဖြစ်မနေဖွင့်ရမယ့် security settings

<Steps>
  <Step title="Strong password သုံးပါ">
    တခြား social account တွေမှာသုံးထားတဲ့ password ကို exchange မှာမသုံးပါနဲ့။ Password manager သုံးနိုင်ရင်ပိုကောင်းပါတယ်။
  </Step>
  <Step title="2FA ဖွင့်ပါ">
    Authenticator app သုံးပါ။ SMS 2FA က SIM swap risk ရှိလို့ primary method မဖြစ်သင့်ပါ။
  </Step>
  <Step title="Passkey ဖွင့်ပါ">
    Exchange က support လုပ်ရင် passkey/FIDO security key ကိုဖွင့်ပါ။
  </Step>
  <Step title="Anti-phishing code ထည့်ပါ">
    Official exchange email တွေမှာ ကိုယ်သတ်မှတ်ထားတဲ့ code ပါမပါစစ်လို့ရအောင်ထည့်ပါ။
  </Step>
  <Step title="Withdrawal whitelist စဉ်းစားပါ">
    Long-term သုံးမယ့် wallet address တွေကို whitelist ထည့်ထားရင် hack ခံရတဲ့အခါ unauthorized withdrawal risk လျော့နိုင်ပါတယ်။
  </Step>
</Steps>

## Login security checklist

Exchange ထဲ login ဝင်တိုင်း:

- URL ကို bookmark ကနေဝင်ပါ။
- Google search ads ကနေမဝင်ပါနဲ့။
- Email link ကို blind မနှိပ်ပါနဲ့။
- Device အသစ်က login alert တွေကိုစစ်ပါ။
- Unknown session တွေကို logout/remove လုပ်ပါ။

## P2P သုံးမယ်ဆိုရင်

P2P မှာ account security အပြင် counterparty risk ပါရှိပါတယ်။

- Verified merchant/reputation စစ်ပါ။
- Receipt screenshot ကို တစ်ခါတည်းမယုံပါနဲ့။
- ကိုယ့် account ထဲ money ရောက်ပြီလား ကိုယ်တိုင်စစ်ပါ။
- Third-party payment မလက်ခံပါနဲ့။
- Chat ကို platform အတွင်းမှာပဲထားပါ။

<Warning>
P2P မှာ “ငွေရောက်ပြီ screenshot ပို့ထားတယ်၊ release လုပ်ပေးပါ” ဆိုတာကို မယုံပါနဲ့။ ကိုယ့် account ထဲ actual balance ရောက်မှ release လုပ်ပါ။
</Warning>

## Account recovery

2FA backup code, password manager master password, email access, phone access တွေကို သီးသန့်ကာကွယ်ထားပါ။ Exchange account ကိုပြန်ရဖို့ email account ကအရေးကြီးပါတယ်။
