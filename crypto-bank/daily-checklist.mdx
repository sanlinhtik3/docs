ဒီ checklist က crypto ကို daily banking tool လိုသုံးတဲ့သူတွေအတွက်ပါ။ Transfer, exchange, P2P, wallet connect မလုပ်ခင် အမြဲပြန်စစ်ပါ။

## Transfer မလုပ်ခင်

- Recipient address မှန်လား
- Network မှန်လား
- Amount မှန်လား
- Fee လက်ခံနိုင်လား
- Memo/tag လိုအပ်လား
- Test transaction လုပ်ပြီးပြီလား

## Wallet connect မလုပ်ခင်

- Website URL မှန်လား
- Official social link ကနေဝင်တာလား
- Wallet request ထဲမှာ ဘာလုပ်ခိုင်းတာလဲ
- Unlimited approval ဖြစ်နေလား
- Main wallet နဲ့မဟုတ်ဘဲ test wallet နဲ့စမ်းထားလား

## Exchange/P2P မလုပ်ခင်

- Counterparty reputation စစ်ပြီးပြီလား
- Platform chat အတွင်းမှာပဲပြောနေလား
- Payment account name တူလား
- Actual bank/mobile money balance ရောက်ပြီးပြီလား
- Release ခလုပ်ကို အမြန်မနှိပ်မိအောင်စောင့်ထားလား

## Weekly routine

တစ်ပတ်တစ်ခါ:

- Exchange login sessions စစ်ပါ။
- Withdrawal whitelist စစ်ပါ။
- Unknown email/login alert ရှိမရှိစစ်ပါ။
- Wallet balances ကို read-only tracker နဲ့စစ်ပါ။
- Scam/phishing news တွေကို update လုပ်ပါ။

## Monthly routine

တစ်လတစ်ခါ:

- Wallet approvals revoke လုပ်ရန်လိုမလိုစစ်ပါ။
- Backup seed phrase ရှိနေသေးလားစစ်ပါ။
- Phone/browser extensions clean လုပ်ပါ။
- Portfolio allocation ပြန်သုံးသပ်ပါ။
