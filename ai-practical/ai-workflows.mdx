AI ကို တစ်ခါမေးတစ်ခါဖြေ chatbot အဖြစ်ပဲသုံးရင် အကျိုးရှိပေမယ့် limit ရှိပါတယ်။ အလုပ်တစ်ခုကို ထပ်ခါထပ်ခါလုပ်ရမယ်ဆိုရင် workflow အဖြစ်တည်ဆောက်တာ ပိုကောင်းပါတယ်။

## Workflow ဆိုတာဘာလဲ

Workflow ဆိုတာ task တစ်ခုကို step တွေအဖြစ်ခွဲပြီး repeatable ဖြစ်အောင်လုပ်ထားတာပါ။

ဥပမာ content workflow:

1. Idea ထုတ်
2. Outline ဆောက်
3. Draft ရေး
4. Fact check
5. Burmese tone polish
6. SEO/frontmatter ထည့်
7. Publish checklist စစ်

## Crypto docs workflow

```md
Input: rough Burmese draft
Step 1: Extract main lesson
Step 2: Turn into docs structure
Step 3: Add risk warnings
Step 4: Add practical checklist
Step 5: Add next lesson links
Output: polished MDX page
```

## AI agent ဆိုတာဘာလဲ

Agent ဆိုတာ goal တစ်ခုပေးလိုက်ရင် tool တွေသုံးပြီး step-by-step လုပ်ဆောင်နိုင်တဲ့ AI workflow ပုံစံပါ။ Agent က powerful ဖြစ်ပေမယ့် guardrails မရှိရင် မှားတဲ့ action လုပ်နိုင်ပါတယ်။

## Beginner rule

AI workflow ကိုအရင်ဆုံး low-risk task တွေနဲ့စမ်းပါ။

- Article outline
- Glossary generation
- Summary
- Checklist
- Translation polish
