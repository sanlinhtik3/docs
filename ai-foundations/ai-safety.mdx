AI က powerful ဖြစ်ပေမယ့် risk မရှိတာမဟုတ်ပါ။ Crypto နဲ့တွဲသုံးရင် ပိုပြီးသတိထားရပါတယ်။

## AI ထဲမထည့်သင့်တဲ့အရာများ

- Seed phrase
- Private key
- Password
- OTP
- API key
- Bank/payment info
- ID card/passport
- Private customer data
- Confidential business data

<Warning>
AI ကို “ငါ့ seed phrase မှန်လားစစ်ပေးပါ” လို့မေးတာက wallet ကိုလက်ဆောင်ပေးလိုက်သလိုဖြစ်နိုင်ပါတယ်။
</Warning>

## Fact checking

AI က source မရှိဘဲ confident ဖြစ်နေနိုင်ပါတယ်။ Crypto price, regulation, product UI, model features, API docs တွေက update ဖြစ်နိုင်တာကြောင့် current source နဲ့ပြန်စစ်ပါ။

## Code safety

AI ရေးပေးတဲ့ code ကို run မလုပ်ခင်ဖတ်ပါ။

- File delete လုပ်နေသလား
- Secret/API key leak ဖြစ်နိုင်လား
- Unknown package install လုပ်ခိုင်းသလား
- Network request ဘယ်ကိုသွားနေလဲ

## Crypto research မှာ AI သုံးရင်

AI ကို buy/sell signal generator အဖြစ်မသုံးပါနဲ့။ AI ကို checklist assistant အဖြစ်သုံးပါ။

Good use:

- Whitepaper summary
- Tokenomics questions
- Risk checklist generation
- Terms explanation

Bad use:

- “ဒီ coin 10x တက်မလား”
- “ဒီနေ့ all-in ဝင်သင့်လား”
- “leverage ဘယ်နှစ်ဆယူရမလဲ”
