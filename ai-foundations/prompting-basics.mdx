Prompt ဆိုတာ AI ကိုပေးတဲ့ instruction ပါ။ Prompt ကောင်းရင် output ကောင်းနိုင်ခြေပိုများပါတယ်။ Prompt မရှင်းရင် AI ကကိုယ်လိုချင်တာကို ခန့်မှန်းပြီးရေးတတ်ပါတယ်။

## Prompt formula

Beginner အတွက် ဒီ formula ကိုသုံးပါ။

```md
Goal:
Audience:
Context:
Task:
Output format:
Constraints:
```

## Example: Crypto article polish

```md
Goal: Myanmar beginner တွေအတွက် DCA article ကိုပိုရှင်းအောင်ပြင်ချင်တယ်။
Audience: Crypto ကိုစလေ့လာတဲ့ Myanmar readers
Context: Friendly Burmese + English tech terms mix tone သုံးချင်တယ်။
Task: အောက်က draft ကို docs style နဲ့ polish လုပ်ပါ။
Output format: headings, steps, warnings, checklist
Constraints: Financial advice မပေးပါနဲ့။ Risk disclaimer ထည့်ပါ။
```

## Example: AI learning assistant

```md
Goal: LLM ဆိုတာကိုနားလည်ချင်တယ်။
Audience: Tech background မရှိတဲ့ beginner
Task: မြန်မာလိုရှင်းပြပါ။
Output format: short explanation, analogy, 5-question quiz
Constraints: English term တွေကို Burmese explanation နဲ့တွဲပေးပါ။
```

## Iteration လုပ်နည်း

Prompt တစ်ခါရေးပြီးမကြိုက်ရင် အောက်ကလိုပြန်ညွှန်ပါ။

- “ပိုတိုအောင်ရေးပါ”
- “Myanmar examples ထည့်ပါ”
- “Beginner အတွက် jargon လျှော့ပါ”
- “Checklist ထည့်ပါ”
- “Risk warning ထည့်ပါ”
- “ဒီ paragraph ကို friendly ဖြစ်အောင်ပြန်ရေးပါ”

## Good prompt checklist

- AI ကို role ပေးထားလား
- Reader ဘယ်သူလဲပြောထားလား
- Draft/context ပေးထားလား
- Output format သတ်မှတ်ထားလား
- မလုပ်စေချင်တာတွေပြောထားလား
- Source/uncertainty rule ထည့်ထားလား

<Tip>
မြန်မာလိုမေးလည်းရတယ်။ Burmese + English tech terms mix သုံးပြီးမေးရင် local docs/content အတွက် ပိုသဘာဝကျတဲ့ output ရနိုင်ပါတယ်။
</Tip>
