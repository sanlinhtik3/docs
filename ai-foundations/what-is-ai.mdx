AI ဆိုတာ computer system တစ်ခုက text, image, audio, code, data တွေကိုနားလည်သလိုလုပ်ပြီး လူကိုကူညီပေးနိုင်တဲ့နည်းပညာအုပ်စုပါ။

ဒီ Docs ထဲမှာ AI ကို theory အနေနဲ့ပဲမဟုတ်ဘဲ နေ့စဉ်အလုပ်မှာသုံးလို့ရတဲ့ assistant အဖြစ်လေ့လာပါမယ်။

## AI ကို ဘာအတွက်သုံးလို့ရလဲ

- စာရှင်းပြခိုင်းတာ
- English/Burmese translation
- Research summary
- Writing/editing
- Coding/debugging
- Data analysis
- Workflow automation
- Idea brainstorming

## AI က လူကိုအစားထိုးတာလား

အလုပ်တချို့ကို AI ကပိုမြန်အောင်လုပ်ပေးနိုင်ပါတယ်။ ဒါပေမယ့် AI ကို blind ယုံလို့မရပါ။ AI က assistant ဖြစ်ပြီး final decision ကို လူကစစ်ရပါမယ်။

## Model နဲ့ tool က ဘာကွာလဲ

- **Model**: Text/image/code generate လုပ်ပေးတဲ့ engine ပါ။
- **Tool**: Model ကို user-friendly ဖြစ်အောင်သုံးခိုင်းတဲ့ app/platform ပါ။

ဥပမာ ChatGPT က tool/platform ဖြစ်ပြီး အထဲမှာ GPT model တွေသုံးနိုင်ပါတယ်။

## AI သုံးတဲ့ beginner rule

- Context ပေးမှ output ကောင်းတတ်ပါတယ်။
- AI ကမှားနိုင်ပါတယ်။
- Private data မထည့်ပါနဲ့။
- Source လိုတဲ့ fact တွေကိုပြန်စစ်ပါ။
- Prompt ကို တစ်ခါရေးပြီးမရရင် iterate လုပ်ပါ။
