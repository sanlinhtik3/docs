Project တစ်ခုရဲ့ဈေးက 10x ဒါမှမဟုတ် 20x ထိတက်ဖို့ဆိုတာက crypto တစ်ခုထဲမှာပဲ ရှိတာပါ။ ဒါကြောင့် လူအများက crypto ကို အတက်ကြမ်း၊ အကျကြမ်းတယ်လို့ ပြောကြတာက မဆန်းတော့ဘူးပေါ့နော်။ ကျွန်တော်တို့ကသာ crypto ဈေးတွေ 10x ထိမတက်ခင် ကြိုသိထားမယ်ဆိုရင် ပိုပြီးတော့ မကောင်းဘူးလားပေါ့နော်။ ဆိုတော့ အဲလို 10x ထိဈေးတွေတက်နိုင်တယ့် crypto တွေကို ဈေးမတက်ခင်မှာ ဘယ်လိုပြုမူတတ်လဲ၊ ပြီးရင် ဘယ်လို ရှာဖွေပြီးတော့ market ထဲကို ကျွန်တော်တို့က စောစောဝင်ထားရင်မကောင်းဘူးလား ဆိုတာကို မျှဝေပေးပါ့မယ်။ အဆင့်သင့်ဖြစ်ပြီဆိုရင်တော့ စလိုက်ကြရအောင်ပါ။

### Project က ဖြေရှင်းမယ့် ပြဿနာနဲ့ ရေရှည်အလားကို စစ်ပါ။

ပထမဆုံးတစ်ခုက Project က ဖြေရှင်းမယ့် ပြဿနာနဲ့ ရေရှည်အလားကို စစ်ပါ။ Project တစ်ခုကို ဖန်တီးပြီဆိုကတည်းက ဘာကြောင့် ဒီ Project ကိုဖန်တီးတာလဲဆိုတဲ့ ရည်ရွယ်ချက်က Project တိုင်းမှာ ရှိပါတယ်။ ဘယ်သူမှ အားအားယားယား ငွေကုန်ကြေးကျ၊ အချိန်ကုန်ခံပြီးတော့ တည်ဆောက်ထားတာ မဟုတ်ဘူးပေါ့နော်။ Project ကိုဖန်တီးတဲ့သူတွေက Project ရဲ့ရည်ရွယ်ချက်၊ ရေရှည်မျှော်မှန်းချက်တွေကို Whitepaper ဆိုတဲ့ထဲမှာ Project တည်ဆောက်ကတည်းက ထည့်ထားပြီးသားပါ။ အဲဒီ Whitepaper ကိုသေချာဖတ်ပြီး ဒီ Project ထဲရင်းနှီးသင့်လား မသင့်ဘူးလား ဆိုတာမျိုးကို ဆုံးဖြတ်ရပါမယ်။

### Project ရဲ့ Team ကိုသေချာစစ်ပါ။

နောက်တစ်ခုက Project ရဲ့ Team ကိုသေချာစစ်ဖို့လိုပါမယ်။ Team မှာပါဝင်သူတွေရဲ့နောက်ခံ သူ့တို့တွေရဲ့ရာဇဝင်တွေဘယ်လိုရှိလဲ၊ အရင်ကရော သူတို့တည်ဆောက်ထားတဲ့ Project တွေက အောင်မြင်မှုတွေရခဲ့လား၊ Team မှာတကယ့်အရည်အချင်းရှိတဲ့သူတွေ ပါရဲ့လားဆိုတာမျိုးကို စစ်ရမှာပါ။ Teamp က နောက်ခံတွေမကောင်းဘူး၊ အရင်လုပ်ခဲ့တဲ့ Project တွေလည်း အောင်မြင်မှုမရဘူး၊ Team ထဲက ကောင်တွေလည်း တစ်ကောင်မှ အရည်အချင်းမရှိဘူးဆိုရင် ကျွန်တော်တို့ကြိုးစားပန်းစားရှာထားတဲ့ ပိုက်ဆံတွေကို Project ထဲထည့်လိုက်ရင် သူတို့အတွက် အလှူငွေထောက်ပံပေးလိုက်သလို ဖြစ်သွားမှာပါ။

### တကယ်ရော Project က အသုံးဝင်ရဲ့လားဆိုတဲ့ use-case ကိုစစ်ပါ။

နံပါတ် (၃) က တကယ်ရော Project က အသုံးဝင်ရဲ့လားဆိုတဲ့ use-case ကိုစစ်ပါ။ တကယ့် Project ရဲ့ ရည်ရွယ်တွေက Whitepaper မှာ အစကတည်းက ပါပြီးသားပါ။ အဲဒီ ရည်ရွယ်ချက်တွေက တကယ်ရောအလုပ်ဖြစ်နေပြီလား၊ လူတွေရဲ့ပြဿနာကို တကယ်ရောဖြေရှင်းပေးနေပြီလား၊ Whitepaper မှာပြောထားတဲ့ အချိန်အတိုင်း လက်ရှိမှာ Project ရဲ့ mindstone တွေ တစ်ခုပြီးတစ်ခု အောင်မြင်နေပြီလား ဆိုတာတွေကို အဓိက စစ်ဖို့လိုပါမယ်။ အဲ့လိုမှမဟုတ်ပဲနဲ့ Project ထဲက admin တွေက x လိုမျိုး socail platform တွေမှာ call တွေပြောလိုက် အာချောင်လိုက်၊ Post တွေရေးလိုက် မျှော်လင့်ချက်တွေပေးလိုက် စသဖြင့် အကြောင်းပြချက်တွေပေးနေပြီဆိုရင် လက်ရှောင်ရပါတော့မယ်။ ညီကိုတို့လည်းသိမှာပါ။ တကယ်ချစ်တဲ့သူက ကိုယ့်ချစ်သူက ဘာကောင်ကြီးပဲဖြစ်ဖြစ်၊ ဘယ်လိုပဲနေနေ၊ ရေမြေတွေခြားပြီးဘယ်လောက်ပဲ ဝေးဝေး ရောက်အောင်လာတွေ့မှာပဲ။ အကြောင်းပြချက်တွေပေးနေပြီဆိုကတည်းက အဆင်မပြေတော့ဘူး။ အဲ့လိုပဲ Project ကိုတစ်ကယ်အကောင်အထည်ဖော်ချင်တဲ့သူတွေက ဘာပဲဖြစ်နေဖြစ်နေ ရအောင်လုပ်မှာပါ။

### Coin ရဲ့ Market cap ကိုစစ်ပါ။

နံပါတ် (၄) က Coin ရဲ့ Market cap ကိုစစ်ပါ။ ဘာကြောင့်လဲဆိုတော့ Market cap သေးငယ်နေတဲ့အခါ ဈေးကွက်ထဲ ရှိသင့်တဲ့ဈေးထက် နည်းနေတာကြောင့် အလွယ်တကူဈေးပြန်တက်ဖို့ ရာခိုင်နှုန်းက များပါတယ်။ ကျွန်တော်က ရေရှည်အလားလားကိုကြည့်ပြီးတော့ အဲဒီ coin ရဲ့ market cap က \$300M အောက်မှာ ရှိနေပြီဆိုရင် ဝယ်ဖို့စဉ်းစားပါတယ်။ ဒါပေမယ့် Project ရဲ့ရေရှည်အလားလာနဲ့ ခုဏက ပြောထားတဲ့ အချက်တွေအားလုံးကိုတိုင်စစ်ပြီး ဒီ project ရဲ့ coin ကို ဝယ်သင့်လားဆိုတာကို ဆုံဖြတ်ရပါမယ်။

### coin တွေကို ဘယ်လိုဘယ်ပုံ ခွဲခြမ်းပြီးတော့ အသုံးပြုထားတာလဲ ကိုစစ်ပါ။

နံပါတ် (၅) ခုမြောက်က project ရဲ့ token တွေကို ဘယ်လိုဘယ်ပုံ ခွဲခြမ်းပြီးတော့ အသုံးပြုထားတာလဲ ကိုစစ်ပါ။ တကယ်တော့ Project စတင်တည်ဆောက်ကတည်းက သက်ဆိုင်ရာ company တွေက ဒီ Project ထဲရင်းနှီးမြှုပ်နှံတာတွေ ရှိပါတယ်။ အဲဒီ ရင်းနှီးမြှုပ်နှံတာတွေနဲ့ Project ရေရှည်ရပ်တည်ဖို့ အသုံးပြုကြပါတယ်။ ဥပမာ Company တွေက \$100M ကို ရင်းနှီးမြှုပ်နှံမယ်ဆိုရင် အဲဒီ \$100M က tokenomic ထဲဝင်သွားပြီး သူတို့ team အတွက်က ဘယ်လောက် % ခွဲတမ်းယူမယ်၊ investor တွေက ဘယ်လောက် % ယူမယ် community တွေက ဘယ်လောက် % ယူမယ်ဆိုတာမျိုးရှိပါတယ်။ အဲ့တာတွေကိုစစ်ပြီး team ဘက်က ပမာဏများများယူထားရင် အဲ့ Project က decentralized မဖြစ်တော့ပဲနဲ့ ဈေးကွက်ကို သူတို့ ကစားချင်သလို ကစားလို့ရနိုင်ပါတယ်။ တကယ်က Community လက်ထဲမှာ tokenomic အရ 70% လောက်ရှိနေမှ တော်သင့်ရာကျပါလိမ့်မယ်။ အဲ့တာတွေကိုလည်း ပြီးခဲ့တဲ့ 2024 တုန်းက telegram tap bot game တွေဖြစ်တဲ့ Not coin, dog coin, haster kombet, x empire တို့တုန်းက တွေ့ကြုံခဲ့ဖူးပြီးသား ဖြစ်မှာပါ။ သူတို့ team တွေဘက်က ဘယ်လောက်ပမာဏကိုင်ထားပြီး market ကို ဘယ်လိုဈေးကစားသွားတာလဲ ဆိုတာတွေကို အားလုံးကြုံခဲ့ဖူးမှာပါ။

### ဘယ်သူတွေက coin တွေကို ဘယ်လောက်ပမာဏအထိ HODL လုပ်ထားလဲ ကိုစစ်ပါ။
