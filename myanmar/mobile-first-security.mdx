Myanmar user တော်တော်များများက phone တစ်လုံးတည်းနဲ့ exchange, wallet, 2FA, social media, email အကုန်သုံးကြပါတယ်။ အဲဒီလိုဆို phone ပျောက်တာ၊ malware ဝင်တာ၊ SIM/account takeover ဖြစ်တာက crypto account တစ်ခုတည်းမဟုတ်ဘဲ ecosystem အကုန်ထိခိုက်နိုင်ပါတယ်။

## Phone lock

- Strong passcode သုံးပါ။
- Biometric သုံးလို့ရပေမယ့် passcode ကိုလည်းကောင်းကောင်းထားပါ။
- Lock screen notification မှာ OTP/email preview မပြအောင်ထားပါ။

## App hygiene

- APK random download မလုပ်ပါနဲ့။
- Unknown keyboard app မသုံးပါနဲ့။
- Screen recorder/overlay permission မလိုတဲ့ app တွေကိုဖယ်ပါ။
- Wallet app official source ကနေပဲ install လုပ်ပါ။

## Email account ကိုကာကွယ်ပါ

Exchange recovery တွေက email ပေါ်မူတည်ပါတယ်။ Email hack ခံရင် exchange account ပါပါသွားနိုင်ပါတယ်။

- Email 2FA ဖွင့်ပါ။
- Recovery email/phone ကို update ထားပါ။
- Password manager သုံးပါ။
- Login alert တွေစစ်ပါ။

## 2FA backup

Authenticator app သုံးရင် backup/recovery ကိုသေချာထားပါ။ Phone ပျောက်ရင် 2FA ပြန်ရဖို့မလွယ်နိုင်ပါ။

<Warning>
2FA code, OTP, recovery code တွေကို screenshot ရိုက်ပြီး gallery ထဲမထားပါနဲ့။
</Warning>

## Travel/public Wi-Fi

Public Wi-Fi ပေါ်မှာ exchange/wallet transaction မလုပ်တာပိုကောင်းပါတယ်။ မဖြစ်မနေသုံးရမယ်ဆိုရင် official app, VPN, device lock, app update တွေသေချာစစ်ပါ။

## Simple setup

Beginner အတွက်:

- Main email တစ်ခုကို exchange/security အတွက်သီးသန့်ထားပါ။
- Daily phone မှာ small amount wallet ပဲထားပါ။
- Long-term wallet seed phrase ကို offline သိမ်းပါ။
- Social media DM link ကနေ wallet မချိတ်ပါနဲ့။
