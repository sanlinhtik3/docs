---
title: "Red Flags Checklist"
description: "Coin, exchange, wallet app, AI trading bot, online community တွေမှာ scam signal ဖြစ်နိုင်တဲ့ red flags."
---

Crypto မှာ red flag တွေကိုမြင်တတ်တာက profit ရှာတာထက် အရေးကြီးပါတယ်။ ပိုက်ဆံမရှုံးတာကလည်း အမြတ်တစ်မျိုးပါ။

## Project red flags

- Guaranteed profit ပြောတယ်။
- Team ဘယ်သူမှမသိရဘူး။
- Whitepaper က copy/paste ဆန်တယ်။
- Product မရှိဘဲ hype ပဲရှိတယ်။
- Token allocation ကိုရှင်းမပြဘူး။
- Liquidity lock မရှိဘူး။
- Admin က question မေးတာကို ban လုပ်တယ်။

## Trading signal red flags

- VIP group ဝင်ရင် 100% win rate ဆိုတယ်။
- Loss screenshot မပြဘူး။
- Borrow money လုပ်ပြီးဝယ်ခိုင်းတယ်။
- All-in ဝင်ခိုင်းတယ်။
- Stop loss/risk plan မရှိဘူး။

## Wallet/app red flags

- Seed phrase တောင်းတယ်။
- APK file ပို့တယ်။
- Official store မဟုတ်တဲ့ link ပေးတယ်။
- Wallet connect လုပ်ပြီး unlimited approval ခိုင်းတယ်။
- Urgent claim timer နဲ့ဖိအားပေးတယ်။

## AI bot red flags

- “AI trading bot guaranteed profit” လို့ပြောတယ်။
- Strategy ကိုမရှင်းပြဘူး။
- Withdrawal မလုပ်ခင် fee ထပ်တောင်းတယ်။
- Real exchange API key ကို withdrawal permission နဲ့တောင်းတယ်။

<Warning>
Exchange API key ပေးရမယ်ဆိုရင် withdrawal permission ကို ဘယ်တော့မှမဖွင့်ပါနဲ့။ Trading bot ကိုသုံးရင်တောင် read/trade-only permission နဲ့သာစဉ်းစားပါ။
</Warning>
